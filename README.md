Things to install:

```
brew install neovim
brew install ripgrep
brew install tmux
brew tap arl/arl
brew install gitmun
brew install television
brew install tmuxinaton
brew install fzf
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

Run `tmux` and press "C-b I" to install tpm plugins.
